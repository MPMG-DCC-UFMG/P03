# Instruções

## Clone o projeto:

```
git clone https://github.com/MPMG-DCC-UFMG/M07
cd M07/search_engine
```

## Setup

```
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
python manage.py migrate
```

## Rode o servidor:

```
python manage.py runserver
````

## Em seguida acesse:

http://127.0.0.1:8000/search/
